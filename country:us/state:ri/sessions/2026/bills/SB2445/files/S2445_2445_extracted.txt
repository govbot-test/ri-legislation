24 SECTION 2. Chapter 37-2 of the General Laws entitled "State Purchases" is hereby
25 amended by adding thereto the following section:
26 37-2-83. Prohibiting state agencies from entering into contracts limiting ability to
27 install or run certain software.
28 A state contract for the licensing of software applications, which are intended to be installed
29 by a state agency and designed to run on generally available desktop or server hardware, shall not
30 limit a state agency's ability to install or run the software on the hardware of the state agency's
31 choosing.
32 SECTION 3. This act shall take effect upon passage.
